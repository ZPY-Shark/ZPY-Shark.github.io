常见目录介绍

  • / - 根目录
  • /root - root用户目录
  • /home/username - 一般用户目录
  • /etc - 配置文件目录
  • /bin - 命令目录
  • /sbin - 管理命令目录
  • /usr/bin /usr/sbin - 系统预装的其他命令

帮助命令

man

manual手册，一共有9章

  • 1 Command 用户命令
  • 2 System Calls 系统调用
  • 3 Library Calls 程序库调用
  • 4 Special Files 特殊文件（设备），/dev目录中的文件
  • 5 File Formats and Conventions 人类可读性的文件格式说明
  • 6 Games 游戏
  • 7 Macro Packages and Conventions 杂项，文件系统描述
  • 8 System Management Commands 系统指令
  • 9 Kernel Routines 内核相关

help

shell 自带的命令为“内部命令”，其他的为“外部命令”

  • 内部命令使用
    help cd

  • 外部命令使用
    ls –help

    如何区分内部命令和外部命令：使用 type 命令

info

info 命令比help命令更详细，但全是英文

文件管理

文件查看

  • pwd

  • cd
    d - ，进入上一次目录

  • ls

    • -l long长格式，显示详细信息
    • -a all全部文件，显示隐藏文件
    • -r reverse逆向，默认使用文件名逆向
    • -t time时间，按时间顺序显示
    • -R recursion递归，显示子文件夹及子文件信息

    可以组合使用，如：ls -lartR

目录的查看与删除

  • mkdir
    mkdir -p /a/b/c, 创建多级目录
  • rm
    • rmdir 删除空目录
    • rm -r 删除非空目录

复制和移动

  • cp 复制文件和目录

    • -r 复制目录
    • -p 保留用户、权限、时间等文件属性
    • -a 等同与-dpr
    • -v 显示复制过程

  • mv 移动或重命名文件

  • 通配符

    • 星号*，匹配多个字符
    • 问号?，匹配单个字符
