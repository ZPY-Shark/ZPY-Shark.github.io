数据结构-链表

发表于 2019-07-18 | 分类于 数据结构与算法
基本数据结构-链表的介绍。

链表（Linked List）

  • 线性结构
  • 并非连续的内存块
  • 通过指针串联各个结点

三种常见的链表结构

  • 单链表
  • 双向链表
  • 循环链表

单链表

  • 每一个结点的结构包含 数据元素 和 指向后继结点点的指针

  • 第一个结点为 头结点，最后一个节点为 尾结点
  • 头结点用来记录链表的基地址，前一个结点指向后一个节点，尾结点指向NULL

优点

  • 高效的插入和删除操作，时间复杂度O(1)

缺点

  • 低效的随机访问，时间复杂度O(n)

循环链表

  • 在单链表的基础上，尾结点不指向NULL，而是指向头结点，构成循环链表

优点

  • 从链尾到链头比较方便

约瑟夫问题

双向链表

  • 每一个结点的结构包含 数据元素、指向前驱结点的指针、指向后继结点的指针

优点

  • 支持双向遍历，更加灵活
  • 在某些情况下，插入、删除比单链表简单高效

缺点

  • 每个结点多占用一些内存空间，存储相同数据量比单向链表更费存储空间
