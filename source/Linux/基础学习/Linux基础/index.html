Linux基础

Linux基础学习。

终端

  • 图形终端
  • 命令行终端
  • 远程终端（SSH VNC）

常见目录介绍

  • / - 根目录
  • /root - root用户目录
  • /home/username - 一般用户目录
  • /etc - 配置文件目录
  • /bin - 命令目录
  • /sbin - 管理命令目录
  • /usr/bin /usr/sbin - 系统预装的其他命令

帮助命令

man

manual手册，一共有9章

  • 1 Command 用户命令
  • 2 System Calls 系统调用
  • 3 Library Calls 程序库调用
  • 4 Special Files 特殊文件（设备），/dev目录中的文件
  • 5 File Formats and Conventions 人类可读性的文件格式说明
  • 6 Games 游戏
  • 7 Macro Packages and Conventions 杂项，文件系统描述
  • 8 System Management Commands 系统指令
  • 9 Kernel Routines 内核相关

help

shell 自带的命令为“内部命令”，其他的为“外部命令”

  • 内部命令使用
    help cd

  • 外部命令使用
    ls –help

    如何区分内部命令和外部命令：使用 type 命令

info

info 命令比help命令更详细，但全是英文

文件管理

文件查看

  • pwd

  • cd
    d - ，进入上一次目录

  • ls

    • -l long长格式，显示详细信息
    • -a all全部文件，显示隐藏文件
    • -r reverse逆向，默认使用文件名逆向
    • -t time时间，按时间顺序显示
    • -R recursion递归，显示子文件夹及子文件信息

    可以组合使用，如：ls -lartR

目录的查看与删除

  • mkdir
    mkdir -p /a/b/c, 创建多级目录
  • rm
    • rmdir 删除空目录
    • rm -r 删除非空目录

复制和移动

  • cp 复制文件和目录

    • -r 复制目录
    • -p 保留用户、权限、时间等文件属性
    • -a 等同与-dpr
    • -v 显示复制过程

  • mv 移动或重命名文件

  • 通配符

    • 星号*，匹配多个字符
    • 问号?，匹配单个字符

文本查看命令

  • cat 文本内容显示到终端
  • head 查看文件开头
  • tail 查看文件结尾，-f同步更新显示
  • wc 统计文件内容信息

打包、压缩、解压缩

  • tar打包命令
    常用参数
    • c - 打包
    • x - 解包
    • f - 指定操作类型为文件
    • v - 显示所有过程

压缩格式

  • gzip 后缀.gz，使用 -z 参数

  • bzip2 后缀.bz2 使用 -j 参数

    双拓展名缩写

  • .tar.gz = .tgz

  • .tar.bz2 = .tbz2

    其他压缩命令，gzip,bzip2,zip

VI编辑器

VI/VIM 编辑器的四种模式

  • 正常模式
  • 插入模式
  • 命令模式
  • 可视模式

正常模式

  • 除小箭头外的上下左右移动光标方式

    • H - 左
    • J - 下
    • K - 上
    • L - 右

  • 复制命令

    • yy - 复制光标所在的整行（包括换行）
    • 数字yy - ‘数字’+yy，复制包括光标所在的一下‘数字’个行（包括换行）
    • y$ - 复制从光标位置到行结尾的字符（不包含换行）

  • 剪切命令（用法与复制一样）

    • dd
    • 数字dd
    • d$

  • 粘贴命令

    • p - 粘贴复制的字符（如果有换行，下面行下移）

  • 撤销命令

    • u - 撤销操作，可撤销多次

  • 重做撤销命令

    • Ctrl+R - 重做撤销的一步，可多次重做

  • 删除单个字符命令

    • x - 删除光标所在位置的单个字符

  • 替换单个字符命令

    • r+’替换字符’ - 用‘替换字符’替换光标所在位置的单个字符

  • 移动光标到所在行

    • ‘行号’+G - 即 ‘行号’+Shift+g
    • g - 移动光标到文本第一行
    • G - 移动光标到文本最后一行
    • ^ - 移动光标到所在行的首部
    • ¥ - 移动光标到所在行的尾部

插入模式

从正常模式进入插入模式的几种做法

  • i - 从当前光标插入
  • I - 从当前光标的行首插入
  • a - 从当前光标的后一个位置插入
  • A - 从当前光标的行尾插入
  • o - 从当前光标的下一行插入新行
  • O - 从当前光标的上一行插入 新行

命令模式

  • 保存

    • ：w ‘filename’ - 保存到文件’filename’
      注意，若VI打开时没有加文件名，则必须输入filename，否则无法保存；若VI打开时有文件名，则只输入w默认保存当打开的文件；若VI打开时有文件名，但输入w+另外一个文件名，则想当与另存为

  • 退出

    • ：q - 退出，wq保存退出，q!不保存退出

  • 执行Linux命令

    • ：!+‘Linux命令’，例如：!ifconfig

  • 查找字符

    • ：/+‘字符’，查找到之后光标移动到第一个找到到位置，使用n查看下一个位置，shift+n查看上一个位置

  • 替换字符

    • ：s/旧字符/新字符 - 例如，将f替换F为 ：s/f/F ，但是默认只替换光标所在行的第一个字符，若光标所在行没有该字符，则提示错误
    • ：%s/旧字符/新字符 - 替换所有行的第一个旧字符
    • ：%s/旧字符/新字符/g - g=global,替换全局的旧字符
    • ：起始行,结束行+以上替换方式，即为在指定起始行到指定行之间进行替换操作

  • VIM设置

    • ：set+设置项 - 但该设置当次有效，若需要永久保存设置，需要修改/etc/vimrc 文件
